Sen2Three Installation

Sen2Three Standalone Tool Installation

Sen2Three binaries can be donwnloaded from
The Software User Manual on this page contains instructions to install the Sen2Three. Follow these instruction before integrating this processor into SNAP.

Sen2Three Integration in SNAP

Once the processor is installed you have to install the plugin. From SNAP, go to Tools/Plugins, under the "Available Plugins" tab the Sen2Three appears (if it doesn't, check your internet connection and your proxy settings from tools/options). Tick the checkbox next to it, click install and follow the instructions.
