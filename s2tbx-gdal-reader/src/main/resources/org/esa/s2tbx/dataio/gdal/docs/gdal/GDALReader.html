Import products using the GDAL library

This command allows to open the data of different products in SNAP. The GDAL library version used is 2.1.0. If there is no suitable reader to open a certain product, then the GDAL reader tries to open the product. If there is at least a suitable reader to open a certain product except the GDAL reader, then a dialog window is displayed to choose the reader used to open the product. If the GDAL reader cannot open the product then a message dialog is displayed.

Most of the supported raster formats are: GeoTIFF, Erdas Imagine, ECW, MrSID, JPEG2000, DTED, NITF, GeoPackage.

For more details please see: GDAL - Geospatial Data Abstraction Library
