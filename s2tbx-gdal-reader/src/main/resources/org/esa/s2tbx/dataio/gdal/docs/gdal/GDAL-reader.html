Import products using the GDAL library

This command allows to import the data of different products in DIMAP format by means of the GDAL library. If there is no specific reader to import a certain product, then the GDAL reader is used by default. The library version used is 2.1.0. Some of the supported raster formats: GeoTIFF, Erdas Imagine, ECW, MrSID, JPEG2000, DTED, NITF, GeoPackage, ...

For more details please see: GDAL - Geospatial Data Abstraction Library
