Import SPOT View (not DIMAP) Products

This command allows to import SPOT-4 and SPOT-5 View products in (older) BIL format (L3 products).

These products have a specific file structure, containing:

  • A metadata file for the actual product, in a non-standard XML format
  • A metadata file, in DIMAP format, for the product that has been used as input for the L3 product
  • A BIL raster file containing the L3 product image
  • A BIL geolayer file containing the latitude and longitude bands that have been used for obtaining the current product
